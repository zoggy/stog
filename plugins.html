Plugins

Some plugins are included in Stog distribution:

asy
inclusion of images produced by the Aysmptote tool
dot
inclusion of Graphs using the Graphiz tools (dot, ...)
extern
use external commands to modify documents during their computation
markdown
using Markdown syntax in Stog documents
multi-doc
definition of several documents into one XML file
nocaml
prevent OCaml code from being evaluated during the compilation of documents
noexec
prevent shell commands from being executed during the compilation of documents
rel-href
change URLs with fragments in href attributes of a document to keep only "#" followed by the fragment, when refering to the same document
sitemap
generate a Sitemap for a web site

Some other plugins are developped separately:

rdf
specify RDF triples within documents, to produce a RDF graph for the whole generated site. It also permits executing Sparql queries to include data in the generated pages
writing
new rewrite rules to use footnotes and bibliographies in your documents

Tell me if you known about other plugins.
