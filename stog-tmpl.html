Stog-tmpl

stog-tmpl is a command-line utility to generate website skeletons. 

Usage: stog-tmpl [options]
where options are:
  --templates  print stog templates directory and exit
  --modules  print stog modules directory and exit
  -d <dir> output to <dir> instead of current directory
  -t <s> set template to use; default is "software"
  -l  list available templates and exit
  --site-title <s> set site title
  --sw-name <name> set name of the software in the generated files
  -help  Display this list of options
  --help  Display this list of options

In the example below, we create a skeleton for the website of a software. The web site title will be "Foobar", and the software name is "foobar": 

mkdir tmp
stog-tmpl \
  -d tmp # output to tmp instead of current directory \
  -t software # use the software template (which is the default) \
  --site-title Foobar # set website title \
  --sw-name foobar # set software name

This result in the following tree of files: 

stog-tmpl-tree
|-- about.html
|-- blog.html
|-- doc.html
|-- download.html
|-- extlink.png
|-- index.html
|-- Makefile
|-- next.png
|-- posts
|   |-- first-post.html
|   `-- release-0.1.0.html
|-- rss.png
`-- style.css

The generated Makefile defines various targets, including:

site
compile the site
test
compile the site in test mode, i.e. using the destination directory as base url
server
run a local server on http://localhost:8080/preview/index.html to get a live preview of the website while you edit the source files.
